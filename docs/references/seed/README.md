# Seed References

This directory stores raw origin transcripts and early product-shaping notes.

These files are provenance, not authoritative specs. Accepted contracts live in `docs/*.md`, and the first implementation slice is constrained by `docs/MVP.md`.

Use seed files to recover rationale, naming history, and long-term product intent. Do not treat every statement in them as current truth; many sections are exploratory, superseded, or future-facing.
